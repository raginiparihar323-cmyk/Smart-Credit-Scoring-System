# Smart Credit Scoring System

AI-powered Credit Scoring Dashboard for predicting customer default risk using Machine Learning.

---

# Project Overview

This project predicts whether a customer is a Good Credit Risk or a Bad / Risky Credit Customer using Machine Learning.

The system analyzes:
- Repayment history
- Billing records
- Payment behavior
- Credit limit
- Financial details

A modern interactive dashboard has been developed using Streamlit for real-time prediction and visualization.

---

# Features

- AI-based Credit Risk Prediction
- Interactive Streamlit Dashboard
- Modern Premium UI
- Risk Probability Gauge
- Real-time Prediction System
- Good / Bad Customer Classification
- Financial Summary Cards

# Machine Learning Model Used

## Random Forest Classifier

Why Random Forest?

- Handles non-linear data effectively
- Works well on tabular financial datasets
- Reduces overfitting
- Provides stable predictions
